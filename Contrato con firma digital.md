#  CONTRATO
## Contrato de Compromiso del Equipo  
### Proyecto: ShareCar – Plataforma Descentralizada de Carpooling  
### Equipo: ShareCar Dev Team

---

## 1. Propósito del Contrato
Este contrato establece los compromisos, responsabilidades y acuerdos formales del equipo durante el desarrollo del proyecto **ShareCar**, una plataforma descentralizada basada en blockchain y contratos inteligentes para el intercambio seguro de vehículos entre usuarios.  
Al aprobar este documento mediante una *pull request*, cada integrante firma digitalmente su compromiso con el proyecto.

---

## 2. Alcance del Proyecto
El equipo se compromete a entregar:

- Implementación del contrato inteligente en Soroban.
- Desarrollo del frontend con integración Web3 y wallet (Freighter).
- Diseño responsivo e intuitivo de la interfaz.
- Sistema de reputación registrado en blockchain.
- Pruebas unitarias completas del contrato inteligente.
- Documentación técnica completa del sistema.
- Actualización frecuente del repositorio con evidencias de progreso.

---

## 3. Entregables Finales del Proyecto

1. Documento de arquitectura del sistema (diagramas de clases, secuencia y estructura general).  
2. Código del contrato inteligente completamente probado.  
3. Interfaz web funcional conectada a la red de pruebas Soroban.  
4. Sistema de pagos automáticos y reputación.  
5. Manual de usuario y manual técnico.  
6. Sprint planning y bitácoras de avances.  
7. Pruebas unitarias y reporte de cobertura.
